## 1.官方新闻  
时隔四年，它又回来了。CitizenCon 将于 10 月 21 日和 22 日在洛杉矶会议中心举行。

这个为期两天的史诗般的活动不仅仅是一次大会，它还是一次泛银河系的庆祝活动，是充满活力的社区为《星际公民》宇宙注入了生命力。在这里，我们可以齐聚一堂，分享我们共同取得的非凡成就，同时展望充满更多奇妙冒险的未来。

系好安全带，飞行员，你即将踏上一段超乎想象的难忘旅程。
